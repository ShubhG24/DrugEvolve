# DrugEvolve: Hybrid Knowledge Graph + LLM System for Drug Repurposing

This repository contains the preprint of the ACL 2025 submission (under review).

## Abstract
Drug repurposing requires synthesizing complex, multi-source biomedical knowledge—both structured and unstructured—into actionable hypotheses. We introduce DrugEvolve, a hybrid system combining Knowledge Graph reasoning (PrimeKG) with Large Language Models for interpretable, literature-grounded drug repurposing.

## Key Contributions
- Graph-based candidate ranking using drug–protein–disease interactions
- LLM-guided biomedical abstract retrieval and synthesis
- Transparent, text-grounded rationale generation
- Clinician-facing real-time inference interface
